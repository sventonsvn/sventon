Searching for entries
The entered search string will be matched against all file and directory names recursively from the current path and below.
Note that the matching is case sensitive!
CamelCase search patterns can also be used.
Searching for CS will for example match
CacheService.java
and
CacheServiceImpl.java.

Searching log messages
The log search handles single or multiple words, phrases, wildcards and boolean operators (AND, OR and NOT). Examples:
  • foo
  • "foo bar"
  • foo NOT bar
  • foo*bar
